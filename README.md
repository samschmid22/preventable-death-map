# Preventable Death Map

## Project Overview

This project analyzes whether premature mortality across U.S. counties can be predicted using public health, socioeconomic, healthcare-access, rurality, and environmental indicators.

The primary analysis uses 2015–2019 pre-COVID mortality data to identify baseline county-level risk patterns. A secondary sensitivity analysis compares these patterns against 2020–2022 pandemic-era mortality using multiple COVID definitions.

## Main Question

Which county-level conditions are most associated with premature death in America, and did those patterns change during the COVID era?

## Project Goal

The goal is to build a public, employer-facing analytics project that demonstrates data acquisition, data cleaning, dataset merging, exploratory analysis, predictive modeling, dashboard development, and clear interpretation.

## Planned Data Sources

* U.S. Census ACS 5-Year
* NCHS Urban-Rural Classification Scheme

## Important Framing

This project analyzes county-level associations and prediction patterns. It does not claim individual-level causation.
